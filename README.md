# Медиа Юг

Простой статический сайт с одной страницей. Сейчас он показывает основное изображение проекта и небольшой экран загрузки, пока картинка не подгрузится.

## Что здесь есть

- `index.html` - главная страница.
- `media_yug.png` - основное изображение.
- `favicon.png` - иконка сайта в браузере.

## Стек

- HTML
- CSS
- Vanilla JavaScript

Здесь нет Maven, Gradle, backend-сервера и сборки.

## Как запустить

Есть два нормальных способа.

### Вариант 1: открыть файл напрямую

Открой `index.html` в браузере. Для этой страницы этого достаточно, потому что проект не использует backend, API-запросы и сборщик.

### Вариант 2: поднять локальный сервер

Перейди в корень проекта:

```bash
cd /Users/nikita/WebstormProjects/media-yug-site
```

Запусти простой HTTP-сервер:

```bash
python3 -m http.server 8000
```

После запуска открой в браузере:

```text
http://127.0.0.1:8000
```

Чтобы остановить сервер, нажми `Ctrl+C` в терминале, где он запущен.

Если порт `8000` занят, можно указать другой порт:

```bash
python3 -m http.server 8080
```

Тогда адрес будет:

```text
http://127.0.0.1:8080
```

## Что нужно для запуска

- Современный браузер.
- Python 3, если хочешь запускать через локальный HTTP-сервер.
- Никаких npm-, Maven- или Gradle-зависимостей не требуется.

## Где менять конфиг

Основные настройки находятся прямо в `index.html`:

- заголовок страницы в `<title>`;
- текст загрузки;
- путь к основному изображению;
- базовые стили.

Если нужно заменить картинку, подложи новый файл вместо `media_yug.png` или поменяй путь в теге `<img>`.

## Ограничения

- Это статическая страница без админки и без API.
- Логика минимальная: страница просто ждёт загрузки изображения и показывает его.
- Если картинка не найдена или путь сломан, страница всё равно откроется, но вместо изображения будет битый ресурс.
