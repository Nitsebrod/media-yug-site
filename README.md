# Медиа Юг

Статическая страница проекта «Медиа Юг».

Сайт показывает одно основное изображение (`media_yug.png`) и экран загрузки, который скрывается после загрузки картинки.

## Файлы проекта

- `index.html` - страница сайта, стили и небольшой скрипт загрузки.
- `media_yug.png` - основное изображение на странице.
- `favicon.png` - иконка вкладки браузера.

## Локальный запуск

Можно просто открыть `index.html` в браузере.

Также можно запустить локальную раздачу файлов из корня проекта:

```bash
python3 -m http.server 8000
```

После запуска сайт будет доступен по адресу:

```text
http://127.0.0.1:8000
```

Остановить локальный сервер можно через `Ctrl+C` в терминале.

Если порт `8000` занят, укажи другой:

```bash
python3 -m http.server 8080
```

И открой:

```text
http://127.0.0.1:8080
```

## Что нужно для запуска

- Браузер.
- Python 3 только для запуска через локальный сервер.

## Где менять содержимое

Основные правки делаются в `index.html`:

- заголовок вкладки;
- текст загрузки;
- путь к изображению;
- стили страницы;
- поведение загрузчика.

Чтобы заменить изображение, можно обновить файл `media_yug.png` или поменять путь в теге `<img>`.

## Ограничения

- Страница не обращается к внешнему API.
- Нет панели управления или формы редактирования.
- При ошибке загрузки картинки страница откроется, но изображение не отобразится.
